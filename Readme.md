Anecdotes App - README (Laravel Backend)
Overview
A full-stack application with React frontend and Laravel backend for sharing programming anecdotes. Features include user authentication, anecdote management, voting system, and infinite scroll loading.

Features
Frontend:

React.js with Vite

React-Bootstrap for UI components

JWT authentication

Infinite scroll pagination

Backend:

Laravel RESTful API

Sanctum for authentication

MySQL database

API resource controllers

Technologies
Frontend: React, Axios, React-Bootstrap, React Router

Backend: Laravel 12, Sanctum, MySQL

Tools: Postman,Composer,xampp
